GGP.org Matches Service

"This box of matches contains a secret life much more intense than that of certain human beings." (Joan Miro)

GGP.org provides a service that developers can use to programmatically publish, discover, and download match data. All of the match data that goes through this service is structured using the ordinary match data format.

Publishing is done by sending a POST request to matches.ggp.org with an AUTH field and a DATA field. The AUTH field is used as a token that authorizes the bearer to publish updates to this match. The DATA field should contain JSON match data, in the format described above. The response from the service will contain the key under which the newly-published match is available.

Every match is given a key K, and can be found at http://matches.ggp.org/K/. For any given match, live updates can be received by subscribing to the syndicated feed at http://matches.ggp.org/K/feed.atom, which is PuSH-enabled (PubSubHubBub).

There are also two global feeds: one which contains all match updates, and one which only contains match completions. Systems can subscribe to these feeds to get live updates about all ongoing or completed matches as they happen. These can be found here:
