General Game Playing


What is general game playing?

General game playing is about playing games that you've never seen before, figuring out the strategy as you go. It can be done by humans or by intelligent machines. Learn more.

How can you play a sample game?

Play games against computer players using the online gaming kiosk. Play games against human opponents using the Artemis gaming server.

How do you develop a computer player?

Check out the GGP Base project to get started building your own player. Register it with the Tiltyard gaming server so it can play matches against other players around the world.

How can you help out?

Explore the ongoing GGP Galaxy project, which aims to bring general game playing to a wider audience on the web.
