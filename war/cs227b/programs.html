These programs have been prepackaged as runnable JAR files. They can run on any machine that has the JVM installed. To get started, just download one of the packages and run it.

Packaged Parametric Player

Configure your own player without having to write a line of code, using the packaged parametric player. Choose the gaming algorithm, manage resource tradeoffs, and control all of the important parameters involved in building a competitive game player.

Packaged Game Server

Once you get your player running, you'll want to have it play games. Use the prepackaged game server to host matches on your machine. Choose games to play from a large online repository, select players to participate in the match, and get started! You can save the match transcript on your computer, or publish it to the web so that everyone can watch the match live. Published matches will count toward an online leaderboard where players are ranked based on the matches they've played.
