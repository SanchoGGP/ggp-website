Developers building general game playing systems can benefit from open source projects hosted by GGP.org: they can build players and servers using the libraries in GGP Base, and they can test their players and publish matches online using GGP Galaxy online services like Tiltyard.

GGP Base

GGP Base is a set of libraries and applications in Java that assist with the development of GGP players by providing a common infrastructure to base work on. GGP Base handles the mechanical aspects of communicating with the match host, parsing GDL, representing the game as a state machine, determining which moves are legal, and so on. Developers using GGP Base can focus on writing innovative gaming algorithms.

GGP Base also provides useful tools for writing games and testing gamers. This includes programs to verify that rulesheets contain legal GDL, programs to visualize matches and let humans easily interact with gamers, programs to automatically run large sets of matches between gamers and analyze the results, and so on. Development, testing, and visualization tools built using the GGP Base can benefit everybody using it.

Get started building your own player using GGP Base using this tutorial.

Want to test out your player? Register it with Tiltyard to play continuously against humans and computers around the world.

Learn more at the GGP Base project page. Questions? Ask on the ggp-base-discuss mailing list.

GGP Galaxy

All of the online services hosted on GGP.org are part of the "GGP Galaxy" project, which aims to bring general game playing to a wider audience through the use of web technologies. That includes this page.

Developers can use the GGP.org services to access large repositories of games, publish matches so they can be viewed online, subscribe to feeds of match results, and so on.

Learn more at the GGP Galaxy project page.
